# AI Gaming Coach

An AI-powered AI Gaming Coach tool built with Next.js 16, TypeScript, Tailwind CSS, and DeepSeek API.

## Features
- Generate AI Gaming Coach content instantly
- Dark gradient UI
- Powered by DeepSeek AI
- Fully responsive

## Setup
```bash
export OPENAI_API_KEY=your_deepseek_api_key
npm install && npm run dev
```
